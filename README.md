## 🚜 What is FarmKit Tools?

FarmKit Tools is a free, open-source project building simple digital tools for small farmers and rural communities who cannot afford expensive agricultural software.

We focus on:
- 🐔 Poultry farmers
- 🐄 Dairy producers
- 🌱 Smallholder agriculture cooperatives

---

## 🛠️ Planned Tools

- **Poultry Feed Calculator** — daily feed needs based on flock size
- **Egg Production Tracker** — simple daily log
- **Milk Yield Calculator** — track dairy output
- **Disease Early Warning Guide** — common symptoms checker
- **Livestock Cost Estimator** — basic expense tracker

---

## 🎯 Our Mission

Millions of small farmers worldwide make decisions without data. We build free tools so every farmer — regardless of income — can access technology that helps them feed their families and communities.
